# Site Style Guide

## Colors
- Background: `#fafafa`
- Text: `#333`
- Link color: `#2c5aa0`
- Link hover: `#1a3f7a`
- Muted text (dates, authors): `#666` or `#777`
- Borders: `#eee`

## Typography
- Primary font: `'Crimson Text', 'Georgia', serif` (body text, weight 300)
- Heading font: `'EB Garamond', 'Garamond', serif` (titles, weight 300-400)
- Base line-height: 1.6

## Layout
- Max width: `1200px` (homepage), `750px` (blog)
- Padding: `20px`

## Header (consistent across all pages)
- Name "Rui Wang" on left (EB Garamond, 2.75em, weight 300)
- Navigation on right: Home, Projects, Blog
- Border bottom: `1px solid #eee`

## Responsive Breakpoints
- Tablet: `max-width: 900px`
- Mobile: `max-width: 600px`
- iPhone optimized: `max-width: 430px`

## Notes
- Always use `#fafafa` background across all pages
- Keep fonts consistent (Crimson Text + EB Garamond)
- Navigation should appear on all pages with same styling
